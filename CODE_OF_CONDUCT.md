# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/),
version 2.1.

In short: be respectful and constructive. Harassment, discrimination, and
personal attacks are not tolerated. Maintainers may remove comments, commits,
issues, and contributors that violate this code.

Report unacceptable behavior to the maintainer via a private GitHub message or a
security advisory. Reports are handled confidentially.
